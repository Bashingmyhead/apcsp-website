Linux Commands

ls -al

  • List all the files in the directory

  • Example uses: when you want to see if a file was removed or not


hostname -I

  • Show the IP address of the machine

  • Example uses: connect your machine to the SSH app through the IP


cd

  • Takes you to the home directory

  • Example uses: for when you want to navigate through your files


mkdir

  • Creates a new directory

  • Example uses: for organizational purposes, like when you want to subdivide file folders


nano [name]

  • Creates new bash script

  • Allows you to access and edit previous scripts

  • Example use: change the code of a procedure script you created

rm [name]

  • Removes specific file

  • Example use: organize your files


./[script name]

  • Executes bash script

  • Example use: the rpistatus bash script we created to ping every device in the room


git init

  • Initializes a Git repository

  • Example use:when you want to push or pull files to or from GitHub

ssh-keygen

  • Generates a RSA key

  • Example use: allows you to publish the public key and keep the private key so you can protect your online repositories

ping [machine name/address]

  • Ping the designated machine and check for packet loss

  • Example use: like in rpistatus, check and see if a machine is alive

cat [name]

  • Allows for the viewing of a file

  • Example use: convenient checking of a file's contents

echo [text]

  • Displays the text following the command

  • Example use: allows for results to be displayed, for instance in the bash script rpistatus
